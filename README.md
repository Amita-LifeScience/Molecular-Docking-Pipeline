## UCSF Chimera 1.19 V. ChimeraX 1.11.1
- Version 1.19 can save structures as .pdbqt files, which are required for docking. 
- Version 1.11.1 cannot save files as .pdbqt.
- Version 1.11.1 has the model panel (left), command box, and reply log (right) in the same window, which makes the user interface better
  - Version 1.19, the reply log appears as a separate window than the one with command box and model panel, hence user has to switch between the two windows
